Retrieval Augmented Generationr

Developed a Retrieval-Augmented Generation (RAG) system in n8n that enables AI-powered question answering using a centralized document knowledge base. The workflow automatically monitors Google Drive for new, updated, and deleted files to keep the knowledge base synchronized. Documents are processed and converted into vector embeddings using Google Vertex AI and stored in Supabase Vector Store for semantic search. An AI Agent retrieves the most relevant information from the vector database before generating responses. This approach improves answer accuracy and reduces AI hallucinations by grounding responses in actual company documents. The solution provides scalable knowledge management and instant access to organizational information.
